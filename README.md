# QuizBiblicoJava

Este projeto é um jogo de perguntas e respostas sobre a Bíblia, desenvolvido 100% para consolidar meu aprendizado em Java.


## Como jogar

1. Execute o programa em um ambiente Java (IDE ou terminal).  
2. Digite `MENU` para acessar o menu do jogo.  
3. Escolha uma das opções:  
   - **1 - Começar o jogo:** responde às perguntas do quiz.  
   - **2 - Pontuação:** mostra quantas perguntas você acertou até agora.  
   - **3 - Sair:** encerra o programa.  
4. Durante o jogo, digite suas respostas conforme solicitado.  

## Tecnologias usadas

- Java  

## Observações

- Este projeto é 100% para aprendizado.  
- Não possui banco de dados ou persistência de pontuação.
